xTuple Extension

  • The Logistify xTuple Extension provides seamless LTL freight integration with xTuple ERP Software.
  • Look for the Logistify 'L' button that will appear automatically for any LTL shipment.
  • With the Logistify extension, comparing quotes from multiple LTL carriers is as easy as clicking the L button next to the 'Ship Via' field.
